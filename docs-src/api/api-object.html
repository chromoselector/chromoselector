The API object

'; $html .= '

'; $html .= 'The purpose of the API object is to shorten the code when '; $html .= 'accessing the ChromoSelector API.

Example of '; $html .= 'basic API usage:'; $html .= '

$(document).ready(function () {
    // Instanciate the color picker
    $("#myColorPicker").chromoselector();
    // Show it
    $("#myColorPicker").chromoselector("show");
    // Change color
    $("#myColorPicker").chromoselector("setColor", "#123456");
    // Log new color to console in HSL format
    console.log($("#myColorPicker").chromoselector("getColor").hsl);
});
'; $html .= '
Same as above, but using the API object:'; $html .= '
$(document).ready(function () {
    // Instanciate the color picker
    var $picker = $("#myColorPicker").chromoselector();
    // Get an instance of the API object
    var api = $picker.chromoselector("api");
    // Now do everything else
    api.show();
    api.setColor("#123456");
    console.log(api.getColor().hsl);
});
'; $html .= '
Most API object function calls can be chained, so the above can be again reduced to the following:'; $html .= '
$(document).ready(function () {
    // Instanciate the color picker and get the API object
    var api = $("#myColorPicker").chromoselector().chromoselector("api");
    // Now do everything else
    console.log(
        api.show().setColor("#123456").getColor().hsl
    );
});
