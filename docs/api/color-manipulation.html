Color Manipulation

ChoromoSelector provides an easy way to manipulate colors in various formats.

Getting a color: 

// Assuming that the plugin has already been instanciated before
var color = $("#myColorPicker").choromoselector("getColor");
You can expect to receive an object like the following: 
{
    hsl: { h: 0, s: 0, l: 0 },
    rgb: { r: 0, g: 0, b: 0 },
    hex: "#000",
    cmyk: { c: 0, m: 0, y: 0, k: 1 },
    setColor: function (color) { ... },
    getTextColor: function () {
        ...
        return ColorObject;
    }
}
Note: that values for hsl, rgb and cmyk colors are all normalised between 0 and 1.

Setting a color: 

// Hex strings, Objects of type Color
// and those representing colors in various
// formats are all acceptable input
$("#myColorPicker").choromoselector("setColor", { h: 0, s: 0, l: 0 });
$("#myColorPicker").choromoselector("setColor", { r: 0, g: 0, b: 0 });
$("#myColorPicker").choromoselector("setColor", "#000");
$("#myColorPicker").choromoselector("setColor", { c: 0, m: 0, y: 0, k: 1 });

// A previously obtained Color object can be reused
var color = $("#myColorPicker").choromoselector("getColor");
// Do something else here ...
...
// Restore the old color
$("#myColorPicker").choromoselector("setColor", color);
// The color can also be reassigned
color.setColor("#a8d");
$("#myColorPicker").choromoselector("setColor", color);
// These are also all valid ways to set a color:
$("#myColorPicker").choromoselector("setColor", color.hsl);
$("#myColorPicker").choromoselector("setColor", color.rgb);
$("#myColorPicker").choromoselector("setColor", color.hex);
$("#myColorPicker").choromoselector("setColor", color.cmyk);

Converting colors:

Note that in these examples the color value of the picker does not change, but we are only using the obtained object for the conversions. 

// Converting from Hex
var color = $("#myColorPicker").choromoselector("getColor");
color.setColor("#ff0000"); // red
console.log(color.hsl)  // prints: { h: 0, s: 1, l: 0.5 }
console.log(color.rgb)  // prints: { r: 1, g: 0, b: 0 }
console.log(color.cmyk) // prints: { c: 0, m: 1, y: 1, k: 0 }


// Converting from Hsl
var color = $("#myColorPicker").choromoselector("getColor");
color.setColor({ h: 0, s: 1, l: 0.5 }); // red
console.log(color.hex)  // prints: "#ff0000"
console.log(color.rgb)  // prints: { r: 1, g: 0, b: 0 }
console.log(color.cmyk) // prints: { c: 0, m: 1, y: 1, k: 0 }

Getting a contrasting color:

You may need a contrasting color if you need to print text on the background of the selected color. Here's how to obtain one: 

$("#textfield").chromoselector("setColor", "#0f0"); // green
var color = $("#textfield").chromoselector("getColor");
color.hex // #0f0
color.getTextColor().hex // #000 (black)
color.getTextColor().getTextColor().hex // #fff (white)



// Change the input textfield colors on update
$("#textfield").chromoselector({
    preview: false,
    update: function() {
        var color = $(this).chromoselector("getColor");
        $(this).css({
            "background-color": color.hex,
            "color": color.getTextColor().hex
        });
    }
});
