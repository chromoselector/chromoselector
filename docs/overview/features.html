Features

  • Stylish Design

    Chromoselector offers a stylish design for the color picker widget that will certainly impress. You won't find anything else like it for your website.

  • Easy To Use

    The minimal configuration for Chromoselector only requires two lines of HTML code and one line of JavaScript code.

  • Customizable

    Chromoselector has 19 properties, 11 events and 12 API methods. This allows for a wide range of customisations to be applied to the color picker in order to cater for your needs. Among other things, you can integrate external controls, specify custom formatting for colors and use an icon as an openers for the picker.

  • Compact

    Chromoselector only requires jQuery and two extra files to work: the color picker JavaScript file and a stylesheet, totalling less than 10kB when gzipped.
