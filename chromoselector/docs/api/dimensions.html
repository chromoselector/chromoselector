Dimensions

  1. Width of the color picker. Set with the width API property.
  2. Height of the preview box. Set in the CSS file.
  3. Width of the hue selector. Set with ringwidth API property.
  4. Width of a channel in the panel. Set with panelChannelWidth API property.
  5. Margin around a channel in the panel. Set with panelChannelMargin API property.

Also, many other dimensions can be changed in the CSS file.
