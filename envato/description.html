Description

The chromoselector color picker is a jQuery plugin that uses the canvas API as a rendering engine. It provides an easy way to add color pickers to forms on your pages, thus allowing users to select colors in many formats in a user-friendly fashion.

Features

  • Stylish Design

    Chromoselector offers a stylish design for the color picker widget that will certainly impress. You won't find anything else like it for your website.

  • Easy To Use

    The minimal configuration for Chromoselector only requires two lines of HTML code and one line of JavaScript code.

  • Customizable

    Chromoselector has 19 properties, 11 events and 12 API methods. This allows for a wide range of customisations to be applied to the color picker in order to cater for your needs. Among other things, you can integrate external controls, specify custom formatting for colors and use an icon as an openers for the picker.

  • Compact

    Chromoselector only requires jQuery and two extra files to work: the color picker JavaScript file and a stylesheet, totalling only 7kB when gzipped.

  • Fast

    Chromoselector has been optimised for speed and will perform well even on devices with limited computational resourses, such as smartphones and tablets.

Documentation

  • Getting Started

    Chromoselector includes a "Getting Started" guide to help you with the integration of the color pickers straight away.

  • API

    The Chromoselector API is publicly disclosed and can be viewed at: http://chromoselector.com/api/. You can therefore see what Chromoselector is capable of before you purchase a licence.

  • Demos

    There are a number of live demos available at http://chromoselector.com/demos/

  • Sandboxes

    Almost every Chromoselector demo comes with a sandbox (jsFiddle), that allows you to play with its settings and code, thus allowing you to get to know the Chromoselector API without purchasing a license.
